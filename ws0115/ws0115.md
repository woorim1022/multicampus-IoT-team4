# ws0115

* 쇼핑몰 ERD 제작(이미지로 제출)
* ERD 바탕으로 DDL문 만들고 데이터 입력 



* 테이블

1. user 
   * userid- PK
   * pwd
   * name
   * age
   * addr
2. item
   * itemid- PK
   * name
   * price
   * 상품등록일자 regdate
3. cart
   * cartid - PK
   * useid(name) - FK
   * itemid(name, price) - FK
4. order
   * orderid - PK
   * userid(name, addr) - FK
   * itemid(name, price) - FK
   * 주문일자 regdate
   * qt
5. favorite
   * favoriteid - PK
   * userid(name) - FK
   * itemid(name, price) - FK
6. rate
   * rateid - PK
   * userid(name) - FK
   * itemid(name) - FK
   * ratenum
